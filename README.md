# alfs_go

ALFS (Automated Linux From Scratch) with go.

alfs_go is based on [LFS-10.0-systemd](http://www.linuxfromscratch.org/lfs/download.html) for x86_64 only.

env:
- ubuntu20.04/deepin v20
- go 1.15
